# Dashboard de Vendas no Excel

## Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de praticar a criação de dashboards no Excel, organizando e transformando dados brutos em informações visuais claras e úteis para análise de vendas e tomada de decisões.

O dashboard foi construído utilizando recursos do Excel voltados para análise de dados e visualização interativa.

---

## Funcionalidades do Dashboard

- Indicadores de vendas
- Filtros interativos
- Visualização gráfica dos dados
- Organização visual para facilitar análises
- Análise de assinaturas Xbox Game Pass

---

## Recursos Utilizados

- Microsoft Excel
- Tabelas Dinâmicas
- Segmentação de Dados
- Gráficos Dinâmicos
- KPIs Visuais

Transformar dados brutos em informações organizadas e visuais, permitindo uma análise eficiente do desempenho de vendas.
